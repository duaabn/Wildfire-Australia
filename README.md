# Wildfire-Australia
Analysis and visualization of wildfire activities in Australia using Pandas, Matplotlib, Seaborn, Folium, and an interactive Dash dashboard for region and year-based insights.

- File: Wildfire_Australia.ipynb

  - Year selection (Dropdown)

  - Bar Chart: Monthly Average Count of Pixels for Presumed Vegetation Fires
